# Degen Token

Degen Token is a token for reward for Degen Gamers

## Description

Degen tokens are token that are used to purchase items in an in-game marketplace while gaming.

## Getting Started

### Installing

* git clone https://github.com/sogobanwo/DegenToken.git
* npm install
